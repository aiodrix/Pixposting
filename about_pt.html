Nosso objetivo é construir uma plataforma simples e prática para a monetização de compartilhamento de links e postagens.

Quando um usuário clica em um link nosso sistema dá um ponto para o usuário que enviou o link ou postagem. Desta forma é possível verificar quais usuários geraram mais engajamento para o site e recopemsá-los

Para começar é muito simples. Basta realizar um PIX no valor de R$ 5,00 para a chave 2nodesw@gmail.com para ter a sua conta verificada. Você não precisa ter uma conta verificada para realizar postagens, mas não terá acesso à monetização, recompensas e outras vantagens.
